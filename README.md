# AnyQuant 迭代二说明


----------
迭代二在迭代一的基础上实现了一些简易的分析功能，允许查看股票的K线图、实时图等历史信息，能够进行自选股票的股票板块地域分析，支持部分因子的比较分析。

迭代二可执行jar包位于**releases**文件夹下，双击即可执行。
本程序需在**网络正常**的环境下使用。
